## Definition

The "**CHECK-LIST – UNIDENTIFIED FLYING OBJECTS**" is the standardized 26-field intake form used by [[entities/project-sign]] (later [[entities/project-grudge]] and Project Blue Book) at [[entities/wright-patterson-afb]]'s Air Materiel Command beginning in 1947–1948. Its 26 fields are the doctrinal ancestor of every modern UAP intake template, including the DoW Misrep / AARO Mission Report format used today.

## Characteristics

The 26 fields covered:

1. Date / 2. Time / 3. Location / 4. Observer name / 5. Occupation / 6. Address / 7. Place of observation / 8. Number of objects / 9. Distance / 10. Time in sight / 11. Altitude / 12. Speed / 13. Direction of flight / 14. Tactics / 15. Sound / 16. Size / 17. Color / 18. Shape / 19. Odor / 20. Apparent construction / 21. Exhaust trails / 22. Weather / 23. Effect on clouds / 24. Sketches / Photographs / 25. Manner of disappearance / 26. Remarks

A typical case = 1 cover Check-List page + 1–4 narrative / witness-statement pages, often with sub-letters (e.g., 48a, 48b, 48c, 48d) for multi-witness incidents tied to a single event.

The form embeds methodology choices: it explicitly requests **observer occupation** (a proxy for trained-witness reliability — see [[concepts/witness-reliability-evaluation]]), **manner of disappearance** (separating fade-outs from instantaneous departures), and **effect on clouds** (a check for genuine atmospheric interaction).
